---------
Accolades
---------

La première accolade doit être positionnée sur la même que l'élément au quel elles appartiennent.

Exemple pour une fonction :

    .. code:: go
        :number-lines:
        :force:

        // Bonne Syntaxe
        func main () {
            // ...
            }

        // Mauviase Syntaxe
        func main ()
        {
            // ...
            }
