####

La Navbar est un élément qui est partagé entre toutes les pages.

    * Exemple de nommage d'un composant "Navbar" : Menu.js

Pour que les liens soient appliqué sur les éléments d'une Navbar, il faut utiliser "Navlink" depuis
la librairie "React-Router-Dom".
