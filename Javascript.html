Intéraction avec l’utilisateur¶

  1. affichage d’un message à l’écran

    alert()

ex:
var myVar = "un message super important";
alert(myVar);

  2. Entrée utilisateur

    prompt()

ex:
var entreeClavier = prompt("tapez du text ici : ");

  3. Confirmation conditionnel

    confirm()

ex:
if (confirm('Voulez-vous exécuter le code JavaScript de cette page ?')) {
    alert('Le code a bien été exécuté !');
    }
/* un Popup doit s'ouvrir et demander de confirmer ([OK]) ou pas ([Annuler])
   la valeur retournée est alors un booléin (true ou false) */

Type¶

  1. Les 3 types de bases

    • number : Ce type contient tous les types numériques ( entier et décimaux)
    • string
    • boolean

  2. Connaitre le type d’une variables

    typeof

ex:
var myVar = 2;
alert(typeof myVar);

  3. Conversion de TYPE

    1. String –> Number

      parseInt()

ex:
var myStr, myNumber;
myStr = "1234";
myNumber = parseInt(myStr);

    2. Number –> String

      ex:
var myNumber, myStr;
myNumber = 1234;
myStr = myNumber + '';

ex: (version simplifiee)
var myVar = 12;
myVar += '';
alert(typeof myVar);

Détails syntaxique¶

Les fonctions¶

  1. Fonctions simples

    // Déclaration
function myFunct(myArg1, myArg2){
            // un super code
            }

// appel
myFunct()

  2. Fonctions anonymes

    // déclaration
function (myArg){
            // un super code
            }

  3. Exécution immédiate d’une fonction, sans appel préalable

    (function (myArg){
            // super code ...
            })();

/* Cette syntaxe permet d'executer du code isolé
   sans appel préallable d'une
   fonction. La fonction anonyme est exécutée automatiquement (et immédiatement)
*/
