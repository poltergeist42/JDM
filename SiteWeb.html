Descriptif:

Ce document présente les grandes étapes pour la création, la mise en place et la publication (en ligne et hors ligne) d’un site web

Préparation¶

  1. Cas d’utilisations (Use cases)

    Permet de définir / Identifier les acteurs (Utilisateurs, Techniciens, autres) du site, puis pour chacun de ces acteurs, de définir les actions qu’ils pourront ou devrons effectuer.

    N.B : il n’est pas nécessaire d’essayer de couvrir tous les cas d’utilisations dès le début. Ils peuvent être enrichie par la suite.

  2. Maquette du site (wireframes)

    Il s’agit de vignette représentant les différents écran du site ainsi liens direct permettant de passer d’un écran à l’autre

    N.B : Il est généralement apprécier de ne pas devoir faire plus de 3 actions pour atteindre un écran.
